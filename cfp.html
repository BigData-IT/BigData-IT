Call For Papers

BigData-IT encourage research submissions in all topics in the intersection of AIOps and big data, including particularly the following broad topics:

  • Knowledge discovery from IT data
  • Data Analytics for IT data
  • Machine learning technology for resource optimization
  • Machine learning for automating IT operations
  • Augmented human intelligence for IT operations
  • Data transformation and integration, especially entity extraction and linking (e.g., ticket data and social media data all pose challenges for data transformation),
and the following specific topics relevant to the recent trends in big data and AI research, where significant challenges require new research:
  • Causal analysis (e.g., analysis of root causes of system failure)
  • Discovery of interpretable and actionable knowledge (e.g., the patterns discovered from IT data need to be interpretable and explainable)
  • Zero-shot and few-shot learning and data mining (e.g., learning based solutions when there is no or very small amount of labeled data)
  • Mining using pre-trained neural language models (e.g., leverage the general knowledge in the pre-trained models)
  • Joint mining of structured and unstructured data (e.g., joint analysis of ticket data and system logs)
  • Human-in-loop data mining (e.g., interactive support for users to diagnose system problems with feedback)
  • Data mining and Machine Learning with feedback (e.g., refine the models using user feedback)
  • Data mining for information/knowledge quality assessment (e.g., assessing quality of knowledge from the web and social media)
We will accept both regular long research papers and short papers to be presented as posters.
